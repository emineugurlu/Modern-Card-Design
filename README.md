📖 Project Description
This project features a modern card design built with HTML and CSS. The layout consists of multiple cards, and when you click on one card:

The clicked card zooms in, revealing additional text.
The other cards get a "distorted" or "shaky" effect, resembling an ant-like movement, adding a dynamic feel to the interaction.
The design is fully responsive and uses CSS transitions and hover effects to create smooth animations.

⚙️ Installation and Usage
Follow the steps below to set up this project locally:

1️⃣ Clone the Project
Clone the project from GitHub using the following command:
git clone https://github.com/emineugurlu/Modern-Card-Design.git

2️⃣ Navigate to the Project Folder
cd Modern-Card-Design

3️⃣ Open and Run the Files
Open the project folder with your code editor (e.g., Visual Studio Code):
code .

4️⃣ Customization (Optional)
If you want to make changes (like card size, colors, text, etc.), feel free to edit the CSS file. You can modify the transform and transition properties to adjust the zoom and animation effects.
